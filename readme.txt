Задачей данной программы было реализовать апроксимацию заданного сигнала, который был предоставлен в виде набора контрольных точек в текстовом файле(input.txt).
Апроксимация производилась методом наименьших квадратов и была реализована с помощью динамической библиотеки, но подключить ее стандартным методом к программ на c# не получилось.
Файл Header.h объявлены те переменные, которые данная динамическая библиотека принимает из программы, к которой подключена.
Файл aproksimatorcc.cpp содержит в себе непосредственно код программы, предназначенный для реализации апроксимации методом наименьших квадратов. По какой-то причине сама c# программа отказывалась видеть в библиотеке больше, чем одну функцию, и при попытке подключить другую из той же библиотеки компилятор выдавал ошибку, поэтому исходный код пришлось немного переделать и для нормальной работы программы использлвать switch. Потери данных при таком варианте не происходит, но код сам выглядит крайне криво.
Файл Form1.cs содержит в себе код программы на с#, в котором по большому счету реализовывалась оболочка пользователя. Частота, окно апроксимции и тд вводятся именно здесь, ну и выводятся результаты программы(максимум или минимум при заданном приходятся на данном окне апроксимации и тд).
